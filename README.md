## KanaDiary

KanaDiary is a beautifully designed, interactive web application for learning Japanese kana. It combines modern UI design with practical learning tools to create an engaging and effective study experience.

Built with simplicity and performance in mind, KanaDiary runs entirely in the browser with no dependencies or setup required.

---

## Features

### Interactive Kana Grid

Explore Hiragana and Katakana through a clean, responsive grid layout. Each character is paired with its romaji for quick reference.

### Pronunciation Guidance

Hover or tap on characters to access detailed pronunciation tips and example words, helping reinforce real-world usage.

### Extensive Theme System

Includes a wide range of carefully crafted dark and light themes. The multi-color palette system creates a visually immersive experience tailored to your preference.

### Typography Variety

Supports over 20 Japanese fonts, allowing learners to familiarize themselves with different writing styles.

### Smooth User Experience

Fluid animations, responsive layout, and intuitive controls ensure a seamless learning environment across devices.

### No Installation Required

Runs entirely in the browser. Just open the HTML file and start learning.

---

## Getting Started

1. Clone the repository:

   ```
   git clone https://github.com/your-username/KanaDiary.git
   ```

2. Open the project folder:

   ```
   cd KanaDiary
   ```

3. Launch the app:

   * Open `index.html` in your browser

No build tools or dependencies are required.

## Design Philosophy

KanaDiary focuses on clarity, accessibility, and engagement. Every interaction is designed to reduce friction and keep learners focused. The interface blends minimalism with expressive visuals to make studying less mechanical and more immersive.
